# Code Blocks

## Code `Support`

### Plain Text

```plain text

```

### Bash

```bash
#!/bin/bash
STR="Hello World!"
echo $STR
```

### C

```c
#include <stdio.h>

int main(void)
{
    printf("hello, world\n");
}
```

```python
print("Hello World")
```

```java
        System.out.println("Hello World!");
    }
}
```

```javascript
console.log("Hello World!");
```

